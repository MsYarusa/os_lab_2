Программу можно запустить следующим образом:

.\test_util(.exe) \<command> \<flag>

Наличие расширение exe зависит от используемой системы. Если Windows то оно есть, если Linux то нет.

В \<command> передаем команду запуска дочернего процесса.

В \<flag> передаем режим запуска:

--bg: запуск в фоновом режиме

--wait: программа будет ожидать завершения дочернего процесса
